用 nohup 最直接。建议直接指定环境里的 Python，可避免 shell 没激活环境。

cd ~/vscode_folder/ENTERPRRISE-GRADE_RAG
mkdir -p logs

nohup /home/lenovo/micromamba/envs/rag-rerank/bin/python scripts/local_reranker_server.py \
  --model-path /home/lenovo/models/bge-reranker-v2-m3 \
  --host 0.0.0.0 \
  --port 8003 \
  > logs/local_reranker.log 2>&1 &

echo $!


看日志：
tail -f logs/local_reranker.log

查进程：
ps -ef | grep local_reranker_server.py

停掉：
pkill -f local_reranker_server.py

**batch 提高（吞吐更重要） embedding（吃显存）

CUDA_VISIBLE_DEVICES=0 \
nohup /home/lenovo/micromamba/envs/rag-embed/bin/python scripts/local_embedding_server.py \
  --model-path /home/lenovo/models/Bge-m3 \
  --host 0.0.0.0 \
  --port 8002 \
  --batch-size 24 \
  --max-length 1024 \
  > logs/local_embedding.log 2>&1 &
